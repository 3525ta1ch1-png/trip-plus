Trip＋（トリッププラス）

目的地からではなく「気分・目的・時間」から旅行先（体験スポット）を探せるアプリ
アプリを開く ワード検索へ

概要

  • 「計画疲れ」「マンネリ化」を解決するため、抽象的な希望（気分・目的・時間）から検索できます。
  • スポットの登録・編集・削除、口コミ、お気に入り機能を実装しています。

主な機能

  • ユーザー認証（ログイン / ログアウト / 新規登録）
  • スポット登録・編集・削除（画像アップロード対応）
  • ワード検索（気分・目的・時間帯・言語）
  • お気に入り登録
  • 口コミ投稿・一覧表示

使用技術

  • Python / Django
  • SQLite（開発）
  • HTML / CSS
※デプロイ後は本番環境に合わせて調整
